# 📊 Dashboard Comercial

Projeto de dashboard interativo para análise de dados comerciais, desenvolvido com **HTML, CSS e JavaScript**, com gráficos dinâmicos e organização modular de dados.

Este projeto foi criado com foco em estudo de **visualização de dados**, **estruturação de dashboard** e **organização de código front-end**, podendo ser usado como base para sistemas maiores.

---

## 🚀 Funcionalidades

- 📈 Gráficos dinâmicos com JavaScript
- 📊 Exibição de métricas comerciais
- 📦 Organização modular de dados
- 🎨 Interface simples e limpa
- ⚡ Atualização automática de informações no dashboard

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Chart.js 
- Estrutura modular JS
